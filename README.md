# 🎵 Análise Exploratória e Insights de Negócio: Spotify Dataset

Este repositório contém a atividade prática desenvolvida para a disciplina de **Introdução à Ciência de Dados** do **CEUB**. O objetivo principal do projeto é explorar um conjunto de dados reais do Spotify, levantar hipóteses, construir visualizações limpas em Python e extrair insights estratégicos de negócio.

## 📊 Perguntas Respondidas & Insights Encontrados

### 1. Quais são os 10 artistas mais populares do dataset?
* **Gráfico:** Barras Horizontais.
* **Insight:** Identifica os artistas com maior apelo de público orgânico na base. Essencial para tomadas de decisão sobre curadoria de grandes playlists e contratação de shows de grande porte.

### 2. Quais são os gêneros musicais mais populares?
* **Gráfico:** Barras Verticais.
* **Insight:** Mapeia as tendências macro do mercado atual. Investir na produção de faixas voltadas aos top 3 gêneros dessa lista reduz o risco financeiro de lançamentos fonográficos.

### 3. Músicas mais agitadas (*energy*) são mais barulhentas (*loudness*)?
* **Gráfico:** Dispersão (Scatter Plot) com linha de tendência.
* **Insight:** Há uma forte correlação linear positiva. Quanto maior a energia da faixa, maior o volume em decibéis, comprovando uma padronização técnica voltada para pistas de dança e academias.

### 4. Qual é a distribuição da duração das músicas?
* **Gráfico:** Histograma.
* **Insight:** A imensa maioria das faixas se concentra rigidamente entre 3 e 4 minutos. Trata-se do "tempo comercial perfeito" do streaming para maximizar repetições (plays) sem entediar o ouvinte.

### 5. Músicas de alta energia/explícitas são mais populares?
* **Gráfico:** Barras Comparativo.
* **Insight:** Músicas com maior intensidade ou letras marcantes registram médias de popularidade mais elevadas, indicando forte consumo e engajamento orgânico do público jovem.

### 6. Como se dividem as músicas em relação ao andamento (BPM)?
* **Gráfico:** Histograma com linha indicativa de média.
* **Insight:** O andamento médio gira em torno de 120 BPM. Essa frequência simula os batimentos cardíacos humanos em momentos de leve agitação, sendo a pulsação ideal para engajar o corpo em movimentos ritmados.
